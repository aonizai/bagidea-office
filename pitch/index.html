The problem

AI agents are powerful.
And completely invisible.

Today's agents live trapped in a chat window or a scrolling terminal. You can't see a team working, can't tell who's doing what, and the moment you close the tab the "office" is gone. Multi-agent work feels like managing ghosts you can't point at.

no presence

A wall of text

One chat thread can't show five agents collaborating. Parallel work collapses into a transcript nobody can follow.

no place

Nowhere to live

Close the window and it's over. Your "AI team" has no home, no continuity, no sense of an ongoing workplace.

no control

Trust, blindly

Agents touch your files and run commands off-screen. Approving actions means reading log lines, not watching the room.

The solution

Give the team a place to work.

BagIdea Office turns your desktop wallpaper into a living HD-2D office. Every character is a real Claude Code agent — give an order to the Director and watch him walk it to the right desk, split the work, and report back. Behind the pixels it's doing real engineering in real folders.

you see it

Work has a body

Agents walk, sit, huddle, and hand off tasks on-screen. Parallel work becomes something you can literally point at.

it's real

Not a screensaver

Each agent is a genuine claude -p session editing files, running tools, and shipping code — the office is just the interface.

it stays

Always at work

It lives on the wallpaper, day and night, picking work back up after a restart or a rate limit. The office never closes.

Under the pixels

One order in. Real work out.

Four layers turn "build me X" into shipped work — every step visible on the wallpaper.

01

The world (Godot HD-2D)

A pixel office on your wallpaper — desks, rooms, day/night that follows your real clock. The stage where work happens.

02

The daemon (local, zero-dependency Node)

Runs on 127.0.0.1. Holds the team registry, routes orders, brokers permissions, streams every event to the world.

03

The agents (real Claude Code)

Each one a headless session with its own persona, skills and tools — working inside your actual project folders.

04

You, anywhere

Command from the wallpaper, the CLI, or Telegram / Discord / LINE. Approvals pop up right where the agent is standing.

Why it's different

Things only an office can do.

parallelism

Ghost clones

An agent splits into translucent clones that work in parallel on a Ghost Deck, then merges the results — visible the whole time.

safety

Spatial permissions

When an agent needs to run something risky, the approval card appears over its head. You allow per-tool, once or forever.

delegation

A real Director

Talk to one manager. He plans, assigns to whoever has the right skills, and walks the result back to your desk.

memory

Learns on the job

Agents pick up skills and keep notes across sessions — the office gets sharper the more you run it.

reach

Hands on the web

Tick one skill and an agent drives a real browser — open pages, click, fill forms, screenshot — watch it or run it quietly.

extensible

Agents build tools

The office writes its own plugins — new panels and commands — then installs them on itself. It grows.

Swappable brains

An ecosystem, not an app

A workplace that extends itself.

plugins

Real apps inside the office

Plugins add panels, routes and agent commands — a music player, a standup board, a live briefing. They pop out into their own resizable windows, like apps in an OS.

the hub

A community marketplace

Browse and one-click-install plugins from a public hub. Anyone can publish; every plugin is a normal GitHub repo from a template.

self-building

The office writes its own

Ask the team for a tool and an agent scaffolds, builds and installs a working plugin — then you keep it or ship it to the hub.

channels

Run it from your phone

Telegram, Discord and LINE connect straight into the office. Assign work from the couch; watch it land on the wallpaper at home.
