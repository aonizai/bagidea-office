# BagIdea Office

**Your desktop wallpaper goes to work.** A living HD-2D office that renders as your
actual desktop wallpaper — behind your icons — where **every character is a real
[Claude Code](https://claude.com/claude-code) session**. Agents walk to their desks,
run real tools in real project folders, hold meetings, and ask for permission with
real prompts. You're the CEO: give an order, a Director delegates to the team, and you
watch the actual work happen.

> **Truth, not theatre.** A glowing screen is a live tool call. A walk to "Security"
> is a real permission request. If it's on screen, it's really happening.

## Install

```sh
npx bagidea-office
```

or install the launcher globally:

```sh
npm i -g bagidea-office
bagidea-office
```

This runs the official installer for your platform (it clones the repo, builds the
Rust shell, fetches the Godot wallpaper engine and wires everything up). After install,
manage the office with the **`bagidea`** command (`start` / `stop` / `restart` / `update`).

- **Windows** — fully supported
- **macOS / Linux** — beta

You'll need [Claude Code](https://claude.com/claude-code) (a Claude subscription or an
API key). Model brains are swappable per agent — Claude, GLM, DeepSeek, Qwen, OpenAI,
Gemini, or local models via Ollama / LM Studio.

## What you get

- A wallpaper office of **real AI agents** doing real work — truth, not theatre.
- **Voice control** (F6 push-to-talk) and control from **Telegram / Discord / LINE**.
- **14 languages**, built for a global release.
- Extensible: a **plugin Hub**, **MCP tools**, an **agent skill library**, and
  **swappable model brains** per agent.
- Free & **open source (MIT)** — and itself built with Claude Code.
