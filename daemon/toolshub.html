MCP คืออะไร? คือวิธีเสริม “ความสามารถใหม่” ให้ทีม AI ที่ Claude ไม่มีมาในตัว — เช่น เปิดเว็บทำงานแทน, อ่าน Gmail, ดูปฏิทิน, คุยกับฐานข้อมูล ฯลฯ แต่ละตัวคือโปรแกรมเล็กๆ ที่รันในเครื่องคุณ แล้ว agent เรียกใช้ได้
  1. กด ＋ เพิ่ม ตัวที่อยากได้ (ตัวที่ต้องใส่ token/path ให้แก้คำสั่งในช่องก่อน)
  2. ไปหน้า ตั้งค่า → agent แล้วติ๊ก tool ที่ขึ้นต้นว่า mcp:<ชื่อ> ให้ agent ที่ต้องการ
  3. เสร็จ! agent ตัวนั้นจะใช้ความสามารถใหม่ได้เลย
⚠️ การเพิ่ม MCP = อนุญาตให้รันโปรแกรมนั้นในเครื่องคุณจริง เพิ่มเฉพาะตัวที่ไว้ใจ (ส่วนใหญ่เป็นของทางการ @modelcontextprotocol/* หรือ @playwright/*)

มีอีกเยอะที่ modelcontextprotocol/servers — ก๊อปคำสั่ง npx … ของตัวไหนมาวางในช่องด้านบนแล้วกดเพิ่มได้เลย · ถ้าตัวไหนเพิ่มแล้ว error ส่วนใหญ่คือชื่อ package เปลี่ยน — เช็คชื่อล่าสุดในลิงก์
